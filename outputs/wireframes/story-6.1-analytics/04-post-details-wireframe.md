## Componentes (Atomic Design)

### Atoms
- **MediaPreview** — Full size media display (photo/video/carousel)
- **SlideNavigation** — Previous/Next slide buttons
- **MetricCard** — Metric value + label + vs average + delta
- **ChartTiny** — Mini line chart (4h, 6h, 12h, 24h)
- **TimelineRow** — Time point + metrics row
- **EngagementType** — Percentage bar + label + count

### Molecules
- **MediaPanel** — Media preview + slide navigation
- **CaptionPanel** — Caption + hashtags + location + mentions
- **EngagementGrid** — 4 MetricCards (Likes, Comments, Saves, Reach)
- **EngagementChart** — Line chart showing saves per hour
- **Comment** — Avatar + username + time + text + likes + reply button
- **InsightBullet** — Icon + text insight

### Organisms
- **PostDetailsHeader** — Title + publication date + back/share buttons
- **MediaSection** — MediaPanel + CaptionPanel side-by-side
- **EngagementSection** — EngagementGrid + EngagementChart
- **TimelineTable** — Rows of data points over time
- **EngagementTypeBreakdown** — 4 percentage bars
- **CommentsSection** — Top comments + view all link
- **InsightsPanel** — 3-4 insights + recommendations
- **PostDetailsPage** — Complete page layout

## Annotations

### Media Panel Behavior
- **Carousel**: Show slide count "4 de 4" with prev/next buttons
- **Video**: Show play button overlay + duration
- **Photo**: Static display
- **Gallery**: Slide indicator dots at bottom (if carousel)
- **Fullscreen**: Click media to open fullscreen view

### Metric Cards
- **Value formatting**: Large numbers
- **Comparison**: vs media average (calculated server-side)
- **Delta**: Arrow + percentage change
- **Color**: Green if above average, gray if below

### Timeline Table
- **Sortable**: Click header to sort by metric
- **Interactive**: Hover row to highlight
- **Timestamps**: Show hours after publication
- **Formatting**: Numbers with thousand separator

### Engagement Breakdown
- **Horizontal bar charts**: 100% stacked
- **Color coded**: Each engagement type has distinct color
- **Labels**: Percentage + absolute count
- **Order**: Likes → Comments → Saves → Shares (by volume)

### Comments Section
- **Show 3 top comments**: Highest engagement first
- **Avatar**: Small circular image (40x40px)
- **Time**: Relative time ("há 2h")
- **Interaction**: Like count + reply button
- **Load more**: "View All Comments →" link

### Insights
- **Auto-generated** from data:
  - Performance vs average
  - Peak engagement time
  - Engagement type breakdown
  - Recommendations
- **Actionable**: Specific next steps

## Accessibility (WCAG AA)

- [ ] Media preview has aria-label describing content
- [ ] Slide navigation buttons have aria-label: "Previous slide", "Next slide"
- [ ] Metric values have aria-label with full context (e.g., "456 likes, 211 more than average")
- [ ] Chart has aria-describedby pointing to summary text
- [ ] Timeline table has proper <th> headers with scope="col"
- [ ] Comments are in list structure (<ol> or <ul>)
- [ ] All links have descriptive anchor text
- [ ] Focus states clearly visible (2px outline)

---

## Mobile Responsiveness

### < 640px
- Media and content stack vertically
- Media width: 100%
- Metric cards: 2x2 grid
- Timeline table: Horizontal scroll
- Comments: Show 1-2, scroll for more
- Chart: Scrollable horizontally

### 640px - 1024px
- Side-by-side layout but narrower
- Media width: 40%
- Metric cards: 2x2 grid
- Chart height: 200px

### > 1024px
- Standard layout (current design)

## Performance Notes

- [ ] Lazy load carousel images (only visible slide + 1 adjacent)
- [ ] Debounce timeline table sorting (300ms)
- [ ] Cache post details API response (15 minutes)
- [ ] Virtual scroll comments if > 100 (unlikely but prepare)
- [ ] Chart: Use lightweight library (Recharts)
- [ ] Preload prev/next post data (pagination)

## Data Requirements

**Post Details:**
```json
{
  "id": "string",
  "media_type": "photo|video|carousel",
  "caption": "string",
  "published_at": "ISO 8601",
  "location": "string (optional)",
  "mentioned_users": ["user1", "user2"],
  "hashtags": ["hashtag1", "hashtag2"],
  "metrics": {
    "likes": number,
    "comments": number,
    "saves": number,
    "shares": number (estimated),
    "reach": number,
    "impressions": number,
    "engagement_rate": number,
    "views": number (for video)
  },
  "timeline": [
    {
      "hours_after": number,
      "likes": number,
      "comments": number,
      "saves": number,
      "reach": number,
      "engagement_rate": number
    }
  ],
  "top_comments": [
    {
      "user": {id, username, avatar},
      "text": string,
      "created_at": ISO 8601,
      "likes": number
    }
  ]
}
```
