# Story 6.4: Engagement Pattern Analysis

**Epic**: 6 | **Priority**: SHOULD | **Owner**: @dev | **Depends On**: 6.3
**Complexity**: 13 pts | **Business Value**: MEDIUM | **Status**: Draft

## 📋 Descripción

Analisar dados históricos de engagement para identificar padrões: qual horário o público engaja mais? Qual tipo de conteúdo (carousel vs reel vs post)? Qual comprimento de caption funciona? Marina usa isso para otimizar estratégia: "publicar sempre às 18h" e "fazer mais carousels".

## 🎯 Critérios de Aceitação

**Given** um perfil com 30+ posts nos últimos 60 dias  
**When** Marina requisita `GET /api/analytics/{profileId}/engagement`  
**Then**:
- [ ] Response inclui top_hours: 3 horários com maior engagement médio
- [ ] Response inclui top_content_types: ranking de tipos (carousel, reel, post, story) por engagement
- [ ] Response inclui trends: engagement está subindo/descendo ao longo do 60 dias
- [ ] Cada insight vem com % de confiança (se só 5 posts, low confidence)

**And** quando filtra por período (últimos 30 vs 60 dias)  
**Then**:
- [ ] Insights mudam baseado no período
- [ ] Trends mostram mudança de padrão entre períodos

## 📐 Escopo Técnico

### Endpoint
```
GET /api/analytics/{profileId}/engagement?days=60
Response: {
  "period_days": 60,
  "top_hours": [
    { "hour": "18:00", "avg_engagement_rate": 12.5, "confidence": 0.95 },
    { "hour": "09:00", "avg_engagement_rate": 10.2, "confidence": 0.92 }
  ],
  "top_content_types": [
    { "type": "carousel", "avg_engagement": 14.3, "posts_count": 12 },
    { "type": "reel", "avg_engagement": 10.1, "posts_count": 8 }
  ],
  "trends": {
    "direction": "up", // "up", "down", "stable"
    "momentum": "+5.2%", // vs previous period
    "recommendation": "Engagement crescendo! Continue a estratégia atual."
  },
  "caption_insights": {
    "avg_length_top_performers": 245, // chars
    "emoji_correlation": 0.82 // posts com emoji têm 82% mais engagement
  }
}
```

### Dependencies
- Story 6.3 (per-post metrics)
- Historical post data (30-100 posts)
- Algorithm: weighted average por hour, sliding window para trends

## 🎯 Business Value

Marina identifica padrões que não vê a olho nu. "Posts de carousel convertems 40% melhor que reels" é actionable insight. Essencial para próximo passo (recomendações). 15% do valor analytics.

## ⚠️ Risks

1. **Correlation ≠ Causation** (Medium) — Pode haver confounders (holidays, trending topics)
   - Mitigação: UI mostra "correlação, não causalidade comprovada"
2. **Low sample size** (Medium) — Novo perfil pode ter padrões falsos por acaso
   - Mitigação: Mostrar confidence scores, recomendar >30 posts antes de usar
