## ⚠️ Riscos Documentados

| Risco | Impacto | Mitigação |
|-------|---------|-----------|
| Docs get out of sync com code | Users confused, support burden | CI gate: auto-generate API docs from annotations |
| Incomplete API documentation | Users cannot use API | Spec generation: enforce 100% coverage check |
| Screenshots outdated | Docs look stale | Use video or animated GIFs instead of static images |
| No schema versioning | Breaking changes unclear | Docs versioned with releases, changelog maintained |
| Low adoption of docs | High support load | Advertise in onboarding + API error messages |
