# Story 1.5: EXA Web Search Integration

**Epic**: 1 | **Priority**: MUST | **Owner**: @dev | **Depends On**: 1.1

## 📋 Descripción

Integrar **EXA MCP** (web search engine) para pesquisa de tendências, análise de concorrentes e busca de conteúdo relevante ao nicho.

## 🎯 Critérios de Aceitação

- [ ] EXA MCP configurado em `.claude/mcp.json`
- [ ] Service `SearchService` implementado
- [ ] Endpoint `POST /api/search` funcionando
- [ ] Busca por nicho/tendências
- [ ] Busca por concorrentes específicos
- [ ] Rate limiting implementado
- [ ] Cache de resultados (avoid duplicate searches)
- [ ] Testes unitários
- [ ] Sem erros de linting

## 📐 Escopo Técnico

### SearchService
```typescript
class SearchService {
  async searchTrends(niche: string): Promise<SearchResult[]>
  async searchCompetitors(handles: string[]): Promise<CompetitorData[]>
  async searchContent(query: string): Promise<ContentIdea[]>
}
```

### Endpoint
**POST /api/search**
```json
Request: { "query": "...", "type": "trends|competitors|content" }
Response: { "results": [...], "cached": false }
```
