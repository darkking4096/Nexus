# Story 4.1: Manual Mode Workflow

**Epic**: 4 | **Priority**: MUST | **Owner**: @dev | **Depends On**: 3.10

## 📋 Descripción

Implementar workflow de Manual Mode: pesquisa → análise → caption → visual → publicação, com aprovação em cada step.

## 🎯 Critérios de Aceitação

- [ ] Endpoint `POST /api/content/generate` (manual mode)
- [ ] Retorna approval panel com cada step
- [ ] User aprova/rejeita/edita
- [ ] Save drafts
- [ ] Testes E2E

## 📐 Escopo Técnico

### Workflow
```
POST /api/content/generate
  ↓ Research
  ← Approve (GET /api/content/{contentId}/approve)
  ↓ Analysis
  ← Approve
  ↓ Caption
  ← Approve/Edit
  ↓ Visual
  ← Approve
  ↓ Publish
  ← Final Approve
```
