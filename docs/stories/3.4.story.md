## 📝 Dev Notes

**Algorithm Details:**
- **Virality Score**: weighted average of (own_engagement_percentile: 40%, competitor_benchmark: 30%, trend_momentum: 30%)
- **Alignment Score**: weighted average of (objective_match: 50%, content_type_fit: 30%, pattern_efficiency: 20%)
- **Opportunities**: compare own metrics vs competitor medians; flag gaps > 20% or trends not yet adopted
- **Content Recommendations**: rank by (virality_potential + alignment_fit), top 3 only

**Error Handling:**
- If no competitor data: use generic benchmarks
- If no history: return "NEW_PROFILE" recommendation set
- Missing objectives: default to generic "maximize_reach"
