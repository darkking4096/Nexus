# Story 2.1: Profile Setup Wizard

**Epic**: 2 | **Priority**: MUST | **Owner**: @dev | **Depends On**: 1.1, 1.2

## 📋 Descripción

Criar wizard interativo para conectar novo perfil Instagram com coleta de informações iniciais (voz, tom, público-alvo, objetivos).

## 🎯 Critérios de Aceitação

- [ ] UI Wizard com 5 steps
- [ ] Step 1: Instagram credentials
- [ ] Step 2: Account validation + data fetch
- [ ] Step 3: Voice & tone definition
- [ ] Step 4: Target audience profiling
- [ ] Step 5: Goals & KPIs
- [ ] Save config to database
- [ ] Validação em cada step
- [ ] Testes E2E
- [ ] Sem erros de linting

### Backend Endpoints
**POST /api/profiles/create**
```json
Request: {
  "instagram_username": "...",
  "instagram_password": "...",
  "display_name": "...",
  "voice_description": "...",
  "tone": "professional|casual|friendly",
  "audience": { "age": "18-35", "interests": [...] },
  "goals": [...]
}
Response: { "profile_id": "..." }
```
