# Story 1.4: Marketing Instagram Squad Integration (Research Module)

**Epic**: 1 | **Priority**: MUST | **Owner**: @dev | **Depends On**: 1.1, 1.2

## 📋 Descripción

Integrar **marketing-instagram-squad** (5 agentes: profile-strategist, content-planner, trend-researcher, copywriter, analytics-agent) para executar fase de **Pesquisa** do pipeline de inteligência.

## 🎯 Critérios de Aceitação

- [ ] Squad carregado em `backend/src/services/ResearchSquad.ts`
- [ ] Endpoint `POST /api/content/research` implementado
- [ ] Análise de concorrentes funcional (lista + top posts)
- [ ] Busca web de tendências (via squad)
- [ ] Análise de histórico do perfil (últimos 30-100 posts)
- [ ] Extração de voz/tom do perfil
- [ ] Output estruturado: JSON com research findings
- [ ] Testes unitários para squad integration
- [ ] Sem erros de linting

## 📐 Escopo Técnico

### ResearchService
```typescript
class ResearchService {
  async research(profileId: string, query?: string): Promise<ResearchOutput>
  // Output: { competitors, trends, history, voice, insights }
}
```

### Endpoint
**POST /api/content/research**
```json
Request: { "profile_id": "..." }
Response: {
  "competitors": [{ "username": "...", "top_posts": [...] }],
  "trends": ["trend1", "trend2"],
  "history": { "top_posts": [...], "patterns": [...] },
  "voice_analysis": { "tone": "...", "audience": "..." },
  "insights": [...]
}
```

### Integração Squad
- `profile-strategist` → Análise profunda de voz/tom
- `trend-researcher` → Pesquisa de tendências (EXA web search)
- `analytics-agent` → Análise de performance histórica
- `content-planner` → Identificar gaps de conteúdo
