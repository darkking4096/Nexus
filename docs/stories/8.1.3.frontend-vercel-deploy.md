## 🧠 Dev Notes

### Current Frontend Context

The frontend is built with React + Vite and located at:
- **Root**: `packages/frontend/`
- **Build Command**: `npm run build` (must work locally first)
- **Output**: `dist/` directory
- **API Calls**: All requests to backend at localhost:5000 (to be updated)

### Deployment Strategy

1. **Phase 1: Local Build Validation** — Ensure `npm run build` succeeds locally
2. **Phase 2: Vercel Project Setup** — Connect GitHub and create Vercel project
3. **Phase 3: Build Configuration** — Configure build command and output directory
4. **Phase 4: Environment Setup** — Add environment variables for all environments
5. **Phase 5: Deployment** — Trigger first deployment and validate
6. **Phase 6: Performance Monitoring** — Measure Core Web Vitals and establish baseline

### Key Technical Decisions

- **Deployment Platform**: Vercel (optimized for Next.js/React, excellent performance)
- **Build System**: Vite (fast, already in use)
- **API Endpoint**: Update to use environment variable (VITE_API_URL)
- **Monitoring**: Vercel Analytics + performance monitoring
- **Regions**: US-East primary, EU fallback
- **Preview Deployments**: Enable for all PRs

### Vite Build Configuration

Frontend uses Vite with:
- **Build Output**: `dist/`
- **Assets**: Served from `/` (root path)
- **Environment Variables**: VITE_* prefix required
- **API URL**: Must be set via environment variable, not hardcoded

### Frontend to Backend Communication

API endpoint configuration must support:
- Local development: `http://localhost:5000`
- Staging: Backend staging URL (TBD by Story 8.1.2)
- Production: Backend production URL

CORS must be configured to allow frontend domain.

### Testing Standards

- **Build Verification**: `npm run build` runs without errors or warnings
- **Type Checking**: `npm run typecheck` passes
- **Linting**: `npm run lint` passes
- **Preview Deployment**: Vercel preview URL is functional
- **API Integration**: API calls to backend successful from preview/production
- **Performance**: Core Web Vitals meet targets
