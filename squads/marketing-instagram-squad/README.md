# 🏗️ Marketing Instagram Squad

**Gerenciamento completo de perfis Instagram com frameworks de especialistas**

---

## 📋 Overview

O **Marketing Instagram Squad** é um sistema inteligente de 5 agentes especializados para gerenciar perfis de Instagram de forma orgânica. Cada agente pensa como um especialista real do marketing, aplicando frameworks comprovados em suas decisões.

### Operação
- **Canal:** Instagram (único)
- **Modo:** Solo (você + agentes)
- **Objetivo:** Crescimento de engajamento, seguidores e conversões

---

## 🎯 Os 5 Agentes

### 1️⃣ **Profile Strategist**
**Define a essência estratégica do perfil**

Pensa como: Alex Hormozi, Donald Miller, Russell Brunson

Frameworks:
- StoryBrand 7-Point (clareza de mensagem)
- Value Equation (maximizar valor percebido)
- Hook-Story-Offer (estrutura irresistível)
- Common Factors Analysis (o que funciona 80%)

**Usa quando:** Analisar perfil novo, definir estratégia, recalibrar após análise de dados

**Saída esperada:** Avatar do cliente, problem arc, value prop, core messages, content pillars, primeiros 5 posts

---

### 2️⃣ **Copywriter**
**Escreve legendas, hooks, CTAs que convertem**

Pensa como: Gary Halbert, Eugene Schwartz, Alex Hormozi

Frameworks:
- 5 Levels of Awareness (unaware → most aware)
- AIDA (Attention → Interest → Desire → Action)
- Curiosity Gaps & Emotional Triggers
- Benefits vs Features
- Value Equation em Copy

**Usa quando:** Escrever captions, hooks, CTAs, textos de stories

**Saída esperada:** Caption pronta, alternativas de CTA, análise de awareness level

---

### 3️⃣ **Content Planner**
**Organiza calendário editorial e briefs de conteúdo**

Pensa como: Gary Vaynerchuk, Russell Brunson, Alex Hormozi

Frameworks:
- Jab Jab Jab Right Hook (80% valor, 20% venda)
- Content Pillars (proporção de temas)
- Soap Opera Sequence (arco narrativo)
- Frequency & Consistency
- Document Don't Create

**Usa quando:** Planejar mês de conteúdo, criar briefs, organizar sequências

**Saída esperada:** Calendário mensal, content map, métricas target, buffer status

---

### 4️⃣ **Trend Researcher**
**Monitora tendências e busca referências virais**

Pensa como: Jonah Berger, MrBeast, Gary Vaynerchuk

Frameworks:
- STEPPS (Social Currency, Triggers, Emotions, Public, Practical, Stories)
- Hook Engineering (pattern interrupt 0-3s)
- Viral Content Analysis
- Niche Virality Detection
- Pattern Recognition

**Usa quando:** Identificar trends, analisar conteúdo viral, sugerir formatos

**Saída esperada:** Top 5 trends, 3 oportunidades, hook recommendations, timing strategy

---

### 5️⃣ **Analytics Agent**
**Mede, analisa e otimiza baseado em dados**

Pensa como: Neil Patel, Alex Hormozi, Jonah Berger

Frameworks:
- Metrics That Matter (vanity vs real metrics)
- AARRR (Acquisition → Activation → Revenue → Retention → Referral)
- A/B Testing Rigor
- Funnel Analysis
- Optimization Cycle

**Usa quando:** Analisar performance, identificar gargalos, rodar experimentos

**Saída esperada:** Performance report, insights, recomendações, experiment plan

## 🚀 Como Usar

### Quick Start (15 min)

```bash
@profile-strategist
Tenho novo perfil de [seu nicho].
Audience é [descrição].
Objetivo é [meta clara].

Defina estratégia completa.
```

### Workflow Típico (5 dias)

**Dia 1 — Strategy**
```bash
@profile-strategist
Novo perfil: [detalhes]
Defina estratégia.
```

**Dia 2 — Copy**
```bash
@copywriter
Escreva 5 captions para os primeiros posts.
Audience: [nível consciência]
Use hooks curiosidade + proof.
```

**Dia 3 — Calendar**
```bash
@content-planner
Crie calendário de abril (4x/semana).
Objetivo: [meta de leads/vendas].
Aplique Jab Jab Right Hook.
```

**Dia 4 — Trends**
```bash
@trend-researcher
Vi esses 3 posts viralizando.
Como adaptamos ao [seu nicho]?
Recomende 2 posts para semana.
```

**Dia 5 — Analytics**
```bash
@analytics-agent
Setup tracking para launch.
Defina métricas primárias.
Crie baseline + experiment plan.
```

## 🎓 Conceitos-Chave

### Profile Context
Cada decisão é alimentada pelo **profile context** — objetivo do perfil, audience, problema a resolver. Garante coerência estratégica.

### Task-First Architecture
Cada agente segue estrutura task-first:
- Input claro
- Framework aplicado
- Output validado
- Checklist seguido

### Common Factors Analysis
Metodologia de otimização:
1. Olhe para top 20%
2. Identifique padrões
3. Teste hipótese
4. Escale o que funciona
5. Repita

## 💡 Dicas

**Para começar:**
- Não tente fazer tudo sozinho. Use os agentes.
- Comece com profile-strategist (alicerce)
- Depois flua pelos outros 4 naturalmente

**Para otimizar:**
- Documente tudo (próxima vez será mais rápido)
- Meça conscientemente (não suposições)
- Adapte os frameworks ao seu contexto

**Para escalar:**
- 1 perfil = aprenda bem, documente patterns
- 2-5 perfis = reutilize estratégias base
- 5+ perfis = crie "templates de nicho"
