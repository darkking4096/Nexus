# Task: Onboard Profile

## Agente
**Profile Strategist**

## Descrição
Coleta e estrutura as informações iniciais de um novo perfil para alimentar a estratégia. Este é sempre o primeiro passo — antes de definir qualquer estratégia, precisamos entender profundamente o negócio, público e posicionamento.

## Inputs Esperados

```
Forneça:
1. Handle do Instagram: @handle_do_perfil
2. Descrição atual do perfil (bio, qual é o nicho?)
3. Qual é o objetivo principal? (seguidores, leads, vendas, awareness?)
4. Qual é a audiência-alvo? (descrição de persona)
5. Tem 5-10 posts já publicados? (envie os links ou screenshots)
6. Tem landing page / site / email list?
7. Qual é a oferta principal? (produto, course, serviço?)
8. Quem são 3 concorrentes ou accounts que admira?
9. Tem dados de analytics hoje? (seguidores, engagement rate)
```

## Steps

### Step 1: Validar Informações
- [ ] Confirmar que todas as 9 perguntas foram respondidas
- [ ] Se faltam dados, fazer perguntas específicas
- [ ] Documentar qualquer restrição (orçamento, idioma, sensibilidades)

### Step 2: Analisar Posts Existentes (se houver)
- [ ] Revisar os 5-10 posts fornecidos
- [ ] Anotar: qual teve melhor engagement? Por quê?
- [ ] Identificar padrões (formato repetido? Tom consistente? Temas recorrentes?)

### Step 3: Estruturar a Persona do Público
- [ ] Usando StoryBrand 7-Point: quem é o HERÓI? (o público-alvo)
- [ ] Qual é o PROBLEMA (frustração, dor, aspiração)?
- [ ] Que RESULTADO ele quer? (transformação desejada)
- [ ] Documentar 3-5 características-chave da persona

### Step 4: Mapear o Posicionamento Competitivo
- [ ] Analisar os 3 concorrentes mencionados
- [ ] Para cada um: o que eles fazem bem? Qual é o estilo deles?
- [ ] Identificar: qual é a DIFERENÇA que este novo perfil pode criar?
- [ ] (ex: menos academic que competitor A, mais entertaining que competitor B)

### Step 5: Estruturar Contexto de Ofertas
- [ ] Entender a oferta principal: o que vende/promove?
- [ ] Se vender: qual é o preço? Qual é a promessa?
- [ ] Se não vender direto: qual é o objetivo? (build audience, lead gen, brand awareness?)
- [ ] Documentar: qual é a métrica que importa mais? (venda, leads, engagement, followers?)

## Output

Entregar um documento estruturado chamado **`{handle}-onboarding-brief.md`** com:

```markdown
# Onboarding Brief — @{handle}

## Informações Básicas
- Handle: @...
- Objetivo Principal: [seguidores | leads | vendas | awareness]
- Data de Criação da Conta: [se conhecer]

## Persona do Público (Herói do StoryBrand)
- **Quem é:** [Descrição de 2-3 frases]
- **Idade/Faixa:** [se aplicável]
- **Problema/Dor Principal:** [o que o incomoda?]
- **Resultado Desejado:** [transformação que busca]
- **Onde se Informa:** [TikTok, YouTube, podcasts, blogs?]

## Posicionamento Competitivo
- **Concorrentes Analisados:** [3 handles]
- **O que Eles Fazem Bem:** [pontos fortes de cada um]
- **Nossa Diferença:** [o que este perfil fará de forma ÚNICA]

## Ofertas e CTAs
- **Oferta Principal:** [descrição]
- **Preço/Barrier (se aplicável):** [preço, email, download?]
- **Métrica que Importa Mais:** [seguidores, leads, $, engagement?]

## Posts Atuais (Análise Rápida)
- **Total de Posts:** [número]
- **Post com Melhor Eng.:** [descrição + métrica]
- **Padrões Identificados:** [formatos que repetem? Temas?]
- **Tons Usados:** [educativo, inspirador, direto, divertido?]

## Recomendações Iniciais
- [ ] Foco em [formato] parece ser mais seguro para este público
- [ ] Principais desafios: [3-5 coisas que precisar resolver]
- [ ] Próximo passo: `define-strategy`

## Meta-informações
- Coletado em: [data]
- Próxima revisão: [data + 30 dias]
```

## Frameworks Usados
- **StoryBrand 7-Point:** Para estruturar a persona (quem é o herói, qual é o problema)
- **Common Factors Analysis:** Para entender o que já funciona nos posts existentes

## Sucesso Significa
✅ Documento completo e claro o suficiente para alimentar a task `define-strategy`  
✅ Profile Strategist tem dados suficientes para descrever avatar, problema e oferta  
✅ Competidores analisados e diferencial identificado
