# Task: Find Trends

## Agente
**Trend Researcher**

## Descrição
Identifica tendências virais no nicho específico e analisa por que estão funcionando. Alimenta @content-planner e @copywriter com oportunidades de conteúdo que "já estão quentes".

## Inputs Esperados

```
Precisa de:
1. {handle}-strategy-document.md (nicho, audience, competitors)
2. Período a analisar (ex: "últimas 2 semanas")
3. Plataformas (ex: "Instagram Reels, TikTok")
4. Competitors/accounts a monitorar (ex: 3-5 handles)
5. Dados históricos (se houver: quais trends capturamos bem antes?)
```

## Steps

### Step 1: Definir Escopo
- [ ] Qual é o nicho exato? (ex: "copywriting education", não só "marketing")
- [ ] Quais são as 5-10 keywords para buscar?
- [ ] Quais plataformas? (Instagram é prioritário, TikTok para inspiração)

### Step 2: Monitorar Contas Top
- [ ] Analisar últimos 10-15 posts dos 3-5 competitors
- [ ] Qual formato está viralizando? (reels > carousels?)
- [ ] Qual tipo de tema? (how-to, story, contraste, resultado?)

### Step 3: Aplicar STEPPS Framework (Jonah Berger)
Para cada trend identificado, analisar:
- [ ] **Social Currency:** Compartilhar faz a pessoa parecer bem/smart?
- [ ] **Triggers:** Qual é o gatilho que lembra as pessoas? (palavra, conceito)
- [ ] **Emotion:** Que emoção o conteúdo ativa? (admiração, riso, fúria, inspiração?)
- [ ] **Public:** As pessoas veem o que outros estão fazendo? (trending hashtags?)
- [ ] **Practical Value:** Há um "hack" ou learning útil?
- [ ] **Stories:** Está embutido em uma narrativa compelente?

**Insight:** Conteúdo vai viral se tem 3+ elementos de STEPPS.

### Step 4: Analisar Hook Engineering (MrBeast)
- [ ] Qual é o "hook" do trend? (o que faz parar o scroll?)
- [ ] É um loop (assista 3 vezes e não cansa)?
- [ ] É previsível ou surpresivo?

### Step 5: Adaptar ao Nicho Específico
- [ ] Trend está em TikTok mas não em Instagram?
- [ ] Podemos adaptar o formato para nosso público?
- [ ] Qual é a versão "Instagram-friendly" deste trend?

### Step 6: Avaliar Pertinência
- [ ] Faz sentido com os core messages?
- [ ] Faz sentido com a oferta?
- [ ] Público-alvo realmente quer isso?
